# Simple Calculator 🧮

Python mein bana hua basic calculator with error handling.

## ✨ Features
- Addition, Subtraction, Multiplication, Division
- Divide by 0 ka error handling
- Invalid input pe crash nahi hota
- Loop mein baar-baar calculation

## 💻 Kaise chalaye
1. `calculator.py` download/run karo
2. Do numbers aur operation (+, -, *, /) daalo
3. Result dekho
4. y/n se continue ya exit karo

## 🛠️ Concepts Used
- Python Functions
- `if-elif-else` statements  
- `try-except` error handling
- `while` loop
- `float()` type conversion
